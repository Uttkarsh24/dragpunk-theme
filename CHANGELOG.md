# Changelog

All notable changes to the **DragPunk Theme** extension will be documented in this file.

## [1.0.0] - 2026-06-30

### Added
- Official launch of **DragPunk** theme.
- **DragPunk Dark**: An obsidian scale-black canvas (`#0c0c0f`) featuring glowing cyber dragon fire red accents (`#ff3366`), dragon hoard gold keywords (`#ffaa00`), and cyber-jade custom React components (`#00ff88`).
- **DragPunk Light**: A highly readable light theme inspired by warm dragon parchment cream (`#fdfbf7`), rich gold, jade, and crimson accents.
- Full UI workbench color specifications for sidebars, activity bars, status bars, peek views, search boxes, and terminal outputs.
- Custom neon bracket-pair colorization guides.
- Highlighting optimizations specifically crafted for MERN developers.
