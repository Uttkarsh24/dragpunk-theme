# DragPunk Theme

**DragPunk** is a premium Visual Studio Code theme extension inspired by the literal combination of **Dragon + Cyberpunk**. Fusing an **obsidian dragon-scale canvas** with **glowing dragon fire red**, **dragon hoard gold**, and **cyber-jade** accents, it is meticulously tailored to enhance code readability for **MERN (MongoDB, Express, React, Node.js) developers**.

---

## Theme Variants

### 1. DragPunk Dark
An obsidian midnight theme representing the scales of a cyber-dragon with glowing crimson highlights, gold keywords, and cyber-jade components.
- **Canvas Background**: `#0c0c0f` (Deep obsidian scale-black)
- **Sidebar & Widgets**: `#08080a` (Shadowed black)
- **Cursor Focus**: `#00ff88` (Cyber Jade Green)
- **Active Tab border / Accent lines**: `#ff3366` (Dragon Fire Red)
- **Selection Highlight**: `#ff336633` (Glowing crimson fire)

### 2. DragPunk Light
A highly legible light theme inspired by warm dragon parchment and fiery accents.
- **Canvas Background**: `#fdfbf7` (Dragon parchment cream)
- **Sidebar & Widgets**: `#f5f2eb` (Bone-gray)
- **Cursor Focus**: `#ff3366` (Fire Red)
- **Active Tab border**: `#ffaa00` (Treasure Gold)

---

## ⚡ Tailored for MERN Developers

DragPunk optimizes coding speed and visual parsing across JavaScript/TypeScript, React JSX, Node, and MongoDB:

*   **React (JSX/TSX)**: Standard HTML tags (`div`, `span`, `p`) are styled in distinct fire-red (`#ff3366` / `#b91c1c`), while **React Custom Components** (`<Header />`, `<Sidebar />`) glow in **Cyber Jade/Teal** (`#00ff88` / `#047857`). JSX props (`className`, `onClick`) stand out in **Flame Orange** (`#ff6b00` / `#c2410c`).
*   **Express & Node.js**: Control structures (`async`, `await`, `import`, `export`, `return`) use bold, glowing **Dragon Hoard Gold** (`#ffaa00` / `#b45309`) to make asynchronous loops and operations immediately identifiable.
*   **MongoDB & JSON**: Document properties and keys stand out in **Cyber Violet** (`#e040fb` / `#7c3aed`) to separate fields from values at a single glance.
*   **Bracket Pair Colorizer**: Matching neon-colored guidelines to help trace nested hooks and functions (`useState`, `useEffect`, express routers) instantly.
